-------------------------------> dotnet commands
1. cd C:\\Codes\\DatingApp
2. dotnet -h
3. dotnet new -h --> new project help
4. dotnet new webapi -h ---> webapi project help
5. dotnet new webapi -n DatingApp.API
6. dotnet run --> run the dotnet program
7. dotnet watch run ---> automatically runs a  application when you save the file with changes.
8. dotnet tool install --global dotnet-ef ---> installing EntityFramework
9. dotnet-ef -h 
10. dotnet-ef migrations -h
11. dotnet-ef migrations add InitialCreate
12. dotnet-ef database update

---------------------------->Angular commands

ng new my-dream-app ---> create an application
cd my-dream-app
ng serve ----> to run a angular app.

Get-ExecutionPolicy -list ---> Check ExecutionPolicy in Powershell for running commands
